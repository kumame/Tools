角色设定生成器

Character Generator

随机生成
参数设置
  • 角色名称 ，身高基数：MIN ，MAX
  • 特征概率： 异瞳 % ，胡须 % ，面痣 % ，面伤 % ，躯体伤 % ，四肢残疾 %
  • 混血概率 %
使用贴士
  • 直接点击坐标图会根据选择的轴进行生成，或是点击随机生成按钮。
  • 身高基数是以纬线 (热带地区) 作为参考，越往寒冷地区 (南北) 越高。
    异色瞳概率会以虚实坐标递增计算，因此不宜设置过大。
    混血在开启种族且高虚构的情况下出现，只以人类为基底进行混血。
  • 角色设定生成器 由 熊豆(kumame) 编写。
